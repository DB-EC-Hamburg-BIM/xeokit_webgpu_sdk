xeokit-sdk is modular:

  • @xeokit-sdk/viewer: Viewer An extensible, high-performance browser-based AECO model viewer.
  • @xeokit-sdk/viewer-xktloader: A viewer plugin to load models from our native XKT format.
  • @xeokit-sdk/viewer-treeview: A viewer plugin that provides a tree view UI to navigate viewer objects.
  • @xeokit-sdk/viewer-navcube: A viewer plugin that provides a 3D cube widget to help navigate the camera.

Getting started

To learn how xeokit-sdk works, see Concepts. To get started developing with the SDK, see SDK Installation below. Find Functions for example scripts created with the SDK already. To use the commandline interface, see Commandline (CLI). If you're interested in contributing to or customizing the project, see contributing.

SDK Installation

Install the core SDK for programmatic use:

npm install --save @xeokit-sdk/core

Then, import some modules:

// ES Modules.
import { Viewer } from '@xeokit-sdk/viewer';

// CommonJS.
const { Viewer } = require('@xeokit-sdk/viewer');
