xeokit

@xeokit

A complete graphics SDK for browser-based AECO model visualization.

Features
  • Browser-based 3D/2D viewer
  • Fast model loading
  • Fast rendering
  • Low memory footprint
  • Multiple canvases
  • Semantic ER data model
  • Pluggable graphics engine (WebGL, WebGPU..)
  • Natively TypeScript
  • Extensible
