Class ViewObject

Represents the visual state of a model object within a View.

Summary
  • Registered by ViewObject.id in View.viewObjects
  • Has a corresponding SceneObject in Scene.sceneObjects
  • May have a corresponding DataObject in DataModel.dataObjects
