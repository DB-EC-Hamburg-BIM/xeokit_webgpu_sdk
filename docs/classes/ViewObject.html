Class ViewObject

A view of an object in a Viewer.

Overview
  • Belongs to a View.
  • Registered by ViewObject.id in View.viewObjects.
  • The ViewObject will also have a corresponding SceneObject in Scene.sceneObjects, and a DataObject in DataModel.dataObjects, all sharing the same ID.
