Class ViewObject

Represents the visual state of a model object within a View.

Summary

Each View will automatically contain a ViewObject for each SceneObject that currently exists. Whenever a SceneObject is created or destroyed, a ViewObject will also automatically be created and destroyed within each View that exists at the time. Each new View we subsequently create will automatically get a ViewObject for each of the SceneObjects that currently exist.

If the SceneObject has a value for SceneObject.viewLayer, then each View will put its corresponding ViewObject in a ViewLayer `

  • Registered by ViewObject.id in ViewLayer.objects
  • Has a corresponding SceneObject in Scene.sceneObjects
  • May have a corresponding DataObject in DataModel.objects
