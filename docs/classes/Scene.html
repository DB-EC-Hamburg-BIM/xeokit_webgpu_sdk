aabb

  • get aabb(): FloatArrayType
    • Defined in src/viewer/scene/Scene.ts:82

    Gets the World-space axis-aligned 3D boundary (AABB) of this Scene.

    The AABB encompases the boundaries of all {@link SceneModels} currently in Scene.models, and is represented by a six-element Float64Array containing the min/max extents of the axis-aligned volume, ie. [xmin, ymin,zmin,xmax,ymax, zmax].

    When the Scene has no content, the boundary will be an inverted shape, ie. [-100,-100,-100,100,100,100].

    Returns FloatArrayType

createModel

  • createModel(params: SceneModelParams): SceneModel
    • Defined in src/viewer/scene/Scene.ts:158

    Creates a new SceneModel within this Scene.

    The SceneModel represents a new model within the Scene and provides an interface through which we can then build geometry and materials within it.

    When we've finished building our SceneModel, we then call SceneModel.finalize, which causes it to immediately begin rendering within all the Views we created previously with Viewer.createView.

    As that happens, each View automatically gets a ViewObject for each of the SceneModel's SceneObjects, to independently represent that SceneObject's visual state in that View.

    When we've finished with the SceneModel, we destroy it using SceneModel.destroy. That will automatically remove its Objects from all our existing Views.

    seealso

    Data.createModel

    Parameters

    • params: SceneModelParams

      SceneModel configuration

    Returns SceneModel
