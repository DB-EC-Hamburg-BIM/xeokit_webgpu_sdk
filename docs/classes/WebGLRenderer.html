Class WebGLRenderer

Creates and renders geometry and materials for a Viewer using WebGL.

You don't normally want to touch this, unless you're configuring a custom allocation/rendering strategy for your Viewer

createModel

  • createModel(params: SceneModelParams): SceneModel

  • Implementation of Renderer.createModel

    • Defined in src/WebGLRenderer/WebGLRenderer.ts:165

    Returns a new SceneModel that will be stored and rendered by this Renderer.

    The SceneModel provides an interface through which we can then build geometry and materials within it. Once we've built the SceneModel and called SceneModel.finalize, the Renderer will immediately begin rendering it all Views that we registered previously with Renderer.registerView.

    When we've finished with the SceneModel, we then call SceneModel.destroy to destroy it.

    Parameters

    • params: SceneModelParams

    Returns SceneModel
