Class WebGLRenderer

Pluggable WebGL-based rendering strategy for a Viewer.

Handles creation and rendering of geometry and materials for a Viewer, using the browser's WebGL 3D graphics API.

Usage

 
import {Viewer, WebGLRenderer} from "@xeokit/viewer";

const myViewer = new Viewer({
    id: "myViewer",
    renderer: new WebGLRenderer({
         textureTranscoder: new KTX2TextureTranscoder({
             transcoderPath: "./../dist/basis/" // <------ Path to BasisU transcoder module
         })
    })
});

createModel

  • createModel(params: SceneModelParams): SceneModel

  • Implementation of Renderer.createModel

    • Defined in webgl/src/WebGLRenderer.ts:195

    Returns a new SceneModel that will be stored and rendered by this Renderer.

    The SceneModel provides an interface through which we can then build geometry and materials within it. Once we've built the SceneModel and called SceneModel.build, the Renderer will immediately begin rendering it all Views that we registered previously with Renderer.registerView.

    When we've finished with the SceneModel, we then call SceneModel.destroy to destroy it.

    Parameters

    • params: SceneModelParams

    Returns SceneModel
