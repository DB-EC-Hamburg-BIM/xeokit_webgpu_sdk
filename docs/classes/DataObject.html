Class DataObject

Metadata about an object within a Viewer.

Overview
  • Belongs to a DataModel
  • Corresponds to a SceneObject when DataObject.id matches the SceneObject.id
  • Stored by SceneObject.id in DataModel.dataObjects and Data.dataObjects
  • Created with DataModel.createDataObject or DataObject.createDataObject
