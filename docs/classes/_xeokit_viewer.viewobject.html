Class ViewObject

Represents the visual state of a ViewerObject in a View.

Summary
  • Stored in View.objects and ViewLayer.objects
  • Viewer automatically creates one of these in each existing View for each ViewerObject created
  • ViewerObject.viewLayerId determines which of the View's ViewLayers to put the ViewObject in

Overview

Every View automatically maintains within itself a ViewObject for each ViewerObject that exists in the Viewer.

Whenever we create a ViewerObject, each View will automatically create a corresponding ViewObject within itself. When we destroy a ViewerObject, each View will automatically destroy its corresponding ViewObject. The ViewObjects in a View are therefore a manifest of the ViewerObjects in the View.
