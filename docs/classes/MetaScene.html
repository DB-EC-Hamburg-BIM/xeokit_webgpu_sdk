Class MetaScene

Provides metadata about the models and objects within a Viewer.

Overview
  • Located at Viewer.metaScene
  • Contains MetaModels to classify the SceneModels in the Viewer's Scene
  • Contains hierarchies of MetaObjects to classify the SceneObjects in the Scene
  • Provides methods to build its MetaModels and MetaObjects
  • Provides methods to iterate and query MetaObjects
