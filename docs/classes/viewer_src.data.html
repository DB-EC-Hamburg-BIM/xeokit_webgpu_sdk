Contains semantic data about the models within a Viewer.

Represents semantic data as a generic entity-relationship graph, with optional sets of properties to attach to entities. This can be used to represent IFC models, as well as any other semantic model that can be represented using an entity-relationship graph. We can also load and merge multiple models, and query them using custom depth-first traversals.

Summary
  • Located at Viewer.data
  • Contains DataModels to classify the SceneModels in the Scene
  • Contains DataObjects to classify the SceneObjects in the Scene
  • Use Data.createModel to create DataModels
  • Use DataModel.createObject to create DataObjects
  • Use DataModel.createPropertySet to create PropertySets
  • Use DataModel.createRelationship to create Relationships between the DataObjects

* ### Example 3. Querying DataObjects

In our third example, we'll extend our previous example to use the <a href="viewer_src.data.html#searchdataobjects">Data.searchDataObjects</a> method to
traverse our data graph and fetch the IDs of the <a href="viewer_src.dataobject.html">DataObjects</a> we find on that path.

One example of where we use this method is to query the aggregation hierarchy of the DataObjects for building
a tree view of an IFC element hierarchy.

````javascript
const objectIds = [];

myViewer.data.searchDataObjects({
    startObjectId: "table",
    includeObjects: [myTypes.FURNITURE_TYPE],
    includeRelated: [myTypes.AGGREGATES_REL],
    objectIds
});

// objectIds == ["table", "tableTop", "redLeg", "greenLeg", "blueLeg", "yellowLeg"];

view.setObjectsHighlighted(objectIds, true);

In our fourth example, we'll extend our previous example to use the Data.searchDataObjects method to traverse our data graph and fetch the IDs of the DataObjects we find on that path.

const table = myViewer.data.objects["table"];

const tableTop = table.related[myTypes[myTypes.AGGREGATES_REL]["tableTop"];
const tableTop = myViewer.data.objects["tableTop"];

const objectIds = [];

    startObjectId: "table",
