Class Data

Contains semantic data about the models within a Viewer.

Summary
  • Located at Viewer.data
  • Contains DataModels to classify the SceneModels in the Scene
  • Contains DataObjects to classify the SceneObjects in the Scene
  • Use Data.createModel to create DataModels
  • Use DataModel.createObject to create DataObjects
  • Use DataModel.createPropertySet to create PropertySets
