Class AmbientLight

An ambient light source within a View.

Overview
  • Has fixed color and intensity that illuminates all objects equally.
  • AmbientLights, DirLights and PointLights are registered by their Component.id on View.lights.

    • cfg: { color?: FloatArrayType; id?: string; intensity?: number } = {}

      AmbientLight configuration

      • Optional color?: FloatArrayType

        RGB color in range [0..1,0..1,0..1]. Default is [0.7, 0.7, 0.7].

      • Optional id?: string

        Optional ID, unique among all components in the parent Scene, generated automatically when omitted.

      • Optional intensity?: number
