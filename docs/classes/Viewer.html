Class Viewer

The viewer at the core of the xeokit SDK.

See main docs page for usage examples.

Overview
  • A browser-based 3D/2D viewer powered by WebGPU and WebGL
  • Has a Scene containing geometry and materials for models
  • Has Data containing general data about models
  • Has Views, which each provide an independent view of the models in a separate canvas
  • Extensible via Plugins
