  • Contains SceneObjects

Abstract createMesh

  • createMesh(cfg: { colors?: FloatArrayType; colorsCompressed?: FloatArrayType; edgeIndices?: number[]; geometryId?: string; id?: string; indices?: number[]; materialId?: string; matrix?: FloatArrayType; normals?: FloatArrayType; normalsCompressed?: FloatArrayType; origin?: FloatArrayType; position?: FloatArrayType; positions?: FloatArrayType; positionsCompressed?: FloatArrayType; positionsDecompressMatrix?: FloatArrayType; primitive?: string; quaternion?: FloatArrayType; rotation?: FloatArrayType; scale?: FloatArrayType; uvs?: FloatArrayType; uvsCompressed?: FloatArrayType }): void
    • Defined in viewer/scene/SceneModel.ts:132

    Creates a mesh within this SceneModel.

    • A mesh can either define its own geometry, or reuse a geometry created previously with SceneModel.createGeometry.
    • A mesh can also be configured with modeling transforms to apply to the geometry.
    • Each mesh can be aggregated into a SceneObject by SceneModel.createSceneObject. Each SceneMesh can belong to a maximum of one SceneObject.

    Parameters

    • cfg: { colors?: FloatArrayType; colorsCompressed?: FloatArrayType; edgeIndices?: number[]; geometryId?: string; id?: string; indices?: number[]; materialId?: string; matrix?: FloatArrayType; normals?: FloatArrayType; normalsCompressed?: FloatArrayType; origin?: FloatArrayType; position?: FloatArrayType; positions?: FloatArrayType; positionsCompressed?: FloatArrayType; positionsDecompressMatrix?: FloatArrayType; primitive?: string; quaternion?: FloatArrayType; rotation?: FloatArrayType; scale?: FloatArrayType; uvs?: FloatArrayType; uvsCompressed?: FloatArrayType }

      Mesh configuration.

      • Optional colors?: FloatArrayType

        Flat array of floating point RGBA vertex colors.

      • Optional colorsCompressed?: FloatArrayType

        Flat array of compressed integer RGBA vertex colors.

      • Optional edgeIndices?: number[]

        Flat array of edge vertex indices.

      • Optional geometryId?: string

        ID of a geometry to use for this mesh. Assumes that the geometry was previously created with SceneModel.createGeometry. If given, the geometry takes precedence over any other geometry parameters given to this function.

      • Optional id?: string

        ID for the mesh, unique within this SceneModel.

      • Optional indices?: number[]

        Flat array of vertex connectivity indices for the geometry primitive type.

      • Optional materialId?: string

        ID of a material to use for this mesh. Assumes that the material was previously created with {@link SceneModel.createMaterial}.

      • Optional matrix?: FloatArrayType

        Mesh modelling transform matrix. Overrides the position, scale and rotation parameters.

      • Optional normals?: FloatArrayType

        Flat array of floating point vertex normals.

      • Optional normalsCompressed?: FloatArrayType

        Flat array of compressed integer vertex normals.

      • Optional origin?: FloatArrayType

        Optional origin, relative to {@link SceneModel.origin}. When this is given, then positions or positionsCompressed are assumed to be relative to this.

      • Optional position?: FloatArrayType

        Local translation.

      • Optional positions?: FloatArrayType

        Flat array of uncompressed floating point vertex positions.

      • Optional positionsCompressed?: FloatArrayType

        Flat array of compressed integer vertex positions. Also requires positionsDecodeMatrix.

      • Optional positionsDecompressMatrix?: FloatArrayType
      • Optional primitive?: string

        Primitive type; Accepted values are 'points', 'lines', 'triangles', 'solid' and 'surface'.

      • Optional quaternion?: FloatArrayType

        Rotation of the mesh, given as a quaternion.

      • Optional rotation?: FloatArrayType

        Rotation of the mesh as Euler angles given in degrees, for each of the X, Y and Z axis.

      • Optional scale?: FloatArrayType

        Local scale.

      • Optional uvs?: FloatArrayType

        Flat array of floating point vertex UV coordinates.

      • Optional uvsCompressed?: FloatArrayType

        Flat array of compressed integer vertex UV coordinates.

    Returns void

Abstract createSceneObject

  • createSceneObject(cfg: { id?: string; meshIds: string[] }): SceneObject
    • Defined in viewer/scene/SceneModel.ts:170

    Creates a SceneObject within this SceneModel.

    • The SceneObject will aggregate meshes created previously with SceneModel.createMesh.
    • The SceneObject will be registered by SceneObject.id in SceneModel.sceneObjects.
    • Automagically gets a ViewObject in each View of the Viewer. The ViewObjects will also get destroyed automagically when this SceneObject is destroyed.

    Parameters

    • cfg: { id?: string; meshIds: string[] }

      SceneObject configuration.

      • Optional id?: string

        ID for the SceneObject, unique within this SceneModel. The SceneObject is registered by this ID in SceneModel.sceneObjects.

      • meshIds: string[]

        IDs of the meshes to aggregate within this SceneObject. Assumes each mesh was created previously with createMesh. Also assumes that each mesh has not already been aggregated by another SceneObject.

    Returns SceneObject
