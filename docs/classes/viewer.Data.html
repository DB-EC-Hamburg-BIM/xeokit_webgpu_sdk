Class Data

Contains semantic data about the models within a Viewer.

Overview
  • Located at Viewer.data
  • Contains DataModels to classify the SceneModels in the Viewer's Scene
  • Contains hierarchies of DataObjects to classify the SceneObjects in the Scene
  • Provides methods to build its MetaModels and dataObjects
  • Provides methods to iterate and query dataObjects
