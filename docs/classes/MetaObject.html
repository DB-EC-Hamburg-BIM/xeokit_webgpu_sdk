Class MetaObject

Metadata about an object within a Viewer.

Overview
  • Belongs to a MetaModel
  • Registered by MetaObject.id in MetaModel.metaObjects and MetaScene.metaObjects
  • Can be connected into parent-child hierarchies
  • Leaf MetaObjects often have corresponding SceneObjects and ViewObjects
  • Created with MetaModel.createMetaObject or MetaObject.createMetaObject
