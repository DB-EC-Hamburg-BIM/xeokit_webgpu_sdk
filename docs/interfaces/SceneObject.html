Interface SceneObject

An object in a SceneModel.

Overview
  • Belongs to a SceneModel, which belongs to a Scene
  • Created with SceneModel.createSceneObject
  • Registered by SceneModel.id in SceneModel.sceneObjects and Scene.sceneObjects
  • Has a corresponding ViewObject in each of the Viewer's Views
  • Can have a corresponding DataObject in the Viewer's Data
