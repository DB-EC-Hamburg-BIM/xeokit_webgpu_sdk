Interface ViewerModel

A model representation within a Viewer.

See Viewer for usage examples.

Summary
  • Stored in Viewer.models
  • Created with Viewer.createModel
  • Contains ViewerObjects
  • Viewer automatically represents each ViewerObject with a corresponding ViewObject in each View
