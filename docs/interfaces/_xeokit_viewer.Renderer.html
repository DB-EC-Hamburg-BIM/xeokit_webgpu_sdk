Interface Renderer

Defines the contract for the rendering strategy used internally within a Viewer.

A Viewer uses an implementation of this to allocate and render geometry and materials using an available browser 3D graphics API, such as WebGL or WebGPU.

Usage

 
import {Viewer} from "@xeokit/viewer";

const myViewer = new Viewer({
    id: "myViewer",
    renderer: new MyRenderer({ })
});
