positionsCompressed

positionsCompressed: IntArrayParam
  • Defined in core/src/GeometryBucketParams.ts:16

GeometryBucketHandle bucket's 3D vertex positions, quantized as 16-bit integers.

Internally, the Viewer uses GeometryCompressedParams.positionsDecompressMatrix to decompress (dequantize) these back to 32-bit floating-point relative-to-center (RTC) positions that are relative to GeometryCompressedParams.origin.

Vertex positions are required for all primitive types.

Optional uvsCompressed

uvsCompressed: IntArrayParam
  • Defined in core/src/GeometryBucketParams.ts:23

GeometryBucketHandle bucket's UV coordinates, quantized as 16-bit integers.

Internally, the Viewer de-quantizes these with GeometryCompressedParams.uvsDecompressMatrix.
