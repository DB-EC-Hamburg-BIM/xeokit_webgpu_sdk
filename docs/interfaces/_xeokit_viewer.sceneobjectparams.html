Optional viewLayerId

viewLayerId: string
  • Defined in viewer/src/scene/SceneObjectParams.ts:25

Causes each View to put the corresponding ViewObject into a ViewLayer with this ID.

When you create a SceneObject, each View will automatically create a ViewObject to represent it. When the SceneObject has a SceneObject.viewLayerId property, then each View will first ensure that it has a ViewLayer with this ID, and will register the ViewObject in that ViewLayer.

Overrides {@link SceneModelParams.viewLayerId}.
