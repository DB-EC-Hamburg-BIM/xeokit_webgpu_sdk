    • Defined in viewer/scene/SceneModel.ts:203

    Creates a mesh within this SceneModel.

    • A mesh can either define its own geometry, or reuse a geometry created previously with SceneModel.createGeometry.
    • A mesh can also be configured with modeling transforms to apply to the geometry.
    • Each mesh can be aggregated into a SceneObject by SceneModel.createSceneObject. Each SceneMesh can belong to a maximum of one SceneObject.

    Parameters

      Mesh configuration.

        Flat array of floating point RGBA vertex colors.

        Flat array of compressed integer RGBA vertex colors.

      • Optional edgeIndices?: IntArrayType

createSceneObject

  • createSceneObject(cfg: { id?: string; meshIds: string[]; transformId?: string }): SceneObject
    • Defined in viewer/scene/SceneModel.ts:247

    Creates a SceneObject within this SceneModel.

    • The SceneObject will aggregate meshes created previously with SceneModel.createMesh.
    • The SceneObject will be registered by SceneObject.id in SceneModel.sceneObjects.
    • Automagically gets a ViewObject in each View of the Viewer. The ViewObjects will also get destroyed automagically when this SceneObject is destroyed.

    Parameters

    • cfg: { id?: string; meshIds: string[]; transformId?: string }

      SceneObject configuration.

      • Optional id?: string

        ID for the SceneObject, unique within this SceneModel. The SceneObject is registered by this ID in SceneModel.sceneObjects.

      • meshIds: string[]

        IDs of the meshes to aggregate within this SceneObject. Assumes each mesh was created previously with createMesh. Also assumes that each mesh has not already been aggregated by another SceneObject.

      • Optional transformId?: string

        Optional ID of a {@link SceneTransform} created previously with createTransform.

    Returns SceneObject

      Texture configuration.

      • Optional buffers?: ArrayBuffer[]

        Transcoded texture data.

      • Optional encoding?: number

        Encoding format. Supported values are LinearEncoding and sRGBEncoding.

      • Optional flipY?: boolean

        Flips this Texture's source data along its vertical axis when true.

      • id: string

        Unique ID for the texture within this SceneModel.

      • Optional image?: HTMLImageElement

        HTML Image object to load into this texture. Overrides src and buffers.

      • Optional magFilter?: number

        How the texture is sampled when a texel covers more than one pixel. Supported values are LinearFilter and NearestFilter.

      • Optional minFilter?: number

        How the texture is sampled when a texel covers less than one pixel. Supported values are LinearMipmapLinearFilter, LinearMipMapNearestFilter, NearestMipMapNearestFilter, NearestMipMapLinearFilter and LinearMipMapLinearFilter.

      • Optional preloadColor?: FloatArrayType

        RGBA color to preload the texture with.

      • Optional src?: string

        Image file for the texture.

      • Optional wrapS?: number

        Wrap parameter for texture coordinate S. Supported values are ClampToEdgeWrapping, MirroredRepeatWrapping and RepeatWrapping.

      • Optional wrapT?: number

        Wrap parameter for texture coordinate T. Supported values are ClampToEdgeWrapping, MirroredRepeatWrapping and RepeatWrapping..
